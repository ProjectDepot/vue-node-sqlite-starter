# Installation

## System Requirements

- Node.js 18+
- npm or yarn

## Clone and Setup

```bash
git clone https://github.com/your-username/vue-node-starter.git
cd vue-node-starter
npm install
```

## Development

```bash
npm run dev
```

Access the app at:

- Frontend: http://localhost:3000
- Backend API: http://localhost:4000

## Production Build

```bash
npm run build
npm start
```
